# 👨‍💻 Damian Ciumacenco — Portfolio

My personal portfolio website built from scratch with HTML, CSS and JavaScript.

🔗 **Live:** [damianciumacenco.github.io](https://damianciumacenco.github.io)

---

## ✨ Features
- Cinematic hero section with robot image and large display typography
- Custom animated cursor with ring follower
- Scroll reveal animations on all sections
- Staggered project card animations
- Parallax hero image on scroll
- Nav hides on scroll down, reappears on scroll up
- Click-to-copy email in contact section
- Fully responsive — mobile friendly
- No frameworks — pure HTML, CSS & JavaScript

---

## 🛠️ Built With
- HTML5
- CSS3 (Custom properties, Flexbox, Grid, Keyframe animations)
- Vanilla JavaScript (IntersectionObserver, scroll events, clipboard API)
- Google Fonts — Bebas Neue, Barlow, Space Mono

---

## 📁 Structure
```
├── index.html    # Main structure and content
├── style.css     # All styling and animations
├── script.js     # Scroll reveal, cursor, parallax, nav
└── hero.png      # Hero robot image
```
